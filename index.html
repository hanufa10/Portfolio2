About Me

I’m a passionate and self-taught front-end web developer with a strong interest in building clean, responsive, and user-friendly websites. I enjoy turning design concepts into functional web interfaces using HTML, CSS, and modern tools. My journey into web development started with curiosity and quickly turned into a dedication to mastering the craft. I’m constantly learning and growing, and I love creating projects that are simple, elegant, and useful. Whether it's a personal portfolio or a reusable UI component, I aim to write code that’s both efficient and easy to maintain.

Projects

Project Title: MindEase

Description: MindEase is a simple, user-friendly mental wellness landing page designed to promote calm and clarity. It features a clean layout with soft colors and smooth transitions, focusing on usability and aesthetic appeal. Whether used for a mindfulness app or a mental health service, MindEase offers a peaceful and professional first impression.

Project Title: Child care system

Description: The Child Care Registration System is a clean, front-end template designed for childcare centers to manage parent signups and child information efficiently. It includes sections for registration forms, program details, and contact information, all styled with a warm, welcoming design. This template is ideal for daycares, preschools, or activity centers looking for a simple and professional online presence.

Project Title: E-Commerce Website

Description: This E-commerce website template is designed for modern online stores, featuring a clean product grid, category filtering, and a responsive layout. It provides an elegant foundation for showcasing items, promoting offers, and guiding customers through a smooth shopping experience. Ideal for fashion, electronics, or small business storefronts.

Project Title: Campus Navigation System

Description: The Campus Navigation Website is a simple, user-focused template built to help students and visitors explore a university or college campus. It features clean layouts, location highlights, and sectioned pages for buildings, departments, and facilities. Perfect for academic institutions aiming to improve wayfinding and accessibility.
